Summary: Latitude vs X

How does latitude affect weather? What aspects of weather does it affect? The answer to these questions may be investigated by analyzing historical weather data from multiple locations around the planet. Understanding is best gained through graphical representation of the results.


Using data available from the OpenWeatherMap API (https://openweathermap.org/api), a dataset was assembbled from 500 randomly selected cities worldwide. Aspects of weather selected for analysis were temperature, cloudiness, wind and humidity. The Lattitude site contains the source data, graphical representations of that data and a few thoughts on trends and correlations observed.
